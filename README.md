   Sistema de Locação de Salas
   Sobre o Projeto

O Sistema de Locação de Salas é uma aplicação desenvolvida com o objetivo de gerenciar o agendamento e a utilização de salas em ambientes corporativos, acadêmicos ou institucionais.

O sistema permite o controle eficiente de reservas, disponibilidade e organização dos espaços, contribuindo para evitar conflitos de horários e melhorar a gestão de recursos físicos — um problema comum em instituições que lidam com múltiplas turmas ou eventos simultâneos .

Objetivo:

Este projeto foi desenvolvido com foco em:

Facilitar o processo de reserva de salas
Evitar conflitos de agendamento
Centralizar informações sobre uso de espaços
Proporcionar uma interface simples e funcional
Servir como aplicação prática para estudo de desenvolvimento de sistemas

 Funcionalidades:
 
   Cadastro e gerenciamento de reservas;
   Cadastro de salas disponíveis;
   Controle de usuários;
   Consulta de disponibilidade de salas;
   Cancelamento e atualização de reservas;
   Organização de horários;
   Tecnologias Utilizadas;


Linguagem:  JavaScript. 
Banco de Dados: MySQL.
Frameworks/Bibliotecas: Node.Express;
IDE: VS Code;
